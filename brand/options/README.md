# Katalogos Brand Options

Same flow as Erevna's brand options. No tenant-continuity constraint here — Katalogos is greenfield, so the brand can be whatever you want.

## How to use

1. Open `wordmark-gallery.html`:
   ```
   file:///C:/desktopContents/projects/SaaS/apps/katalogos-web/brand/options/wordmark-gallery.html
   ```
2. Browse 12 wordmarks + 15 taglines.
3. Tell me your picks by ID:
   - Wordmark: e.g. `KW-05` (IBM Plex Sans)
   - Tagline: e.g. `KT-05` ("Set the table.")
4. I'll generate the wordmark SVG and a landing-hero preview, then move to the next brand step (palette, typography, voice, marketing copy).

## ID prefixes

- `KW-XX` = **K**atalogos **W**ordmark
- `KT-XX` = **K**atalogos **T**agline

(Erevna's were `W-XX` and `T-XX` so cross-referencing across the two galleries doesn't collide.)

## Greek script support

Same as Erevna's gallery — fonts marked with **GREEK ✓** correctly render Κατάλογος; others fall back to system Greek font.
