## 1. Problem

## 3. Architecture Decisions

- **Preferences storage**: New PostgreSQL table (not Keycloak attributes) for proper indexing, types, and tenant isolation
- **Password change**: Validate old password via ROPC flow, then reset via Admin API
- **Endpoint structure**: Separate `Me/` folder from admin `Users/` folder
- **Self-service auth**: All Me/ endpoints use `Roles(Admin, User)`, NOT Administrator
- **User ID**: Extracted from JWT `sub` claim (no IDOR risk)
