# AI Menu Import Frontend UI

## Status: COMPLETED

## Problem Statement
Users should be able to import menus by uploading photos/PDFs of physical menus. The backend AI endpoints extract categories and items. The frontend needs a wizard modal to upload, review extracted data, edit it, and apply it to a menu.

## Backend Endpoints
- `POST /api/v1/TenantMenus/import-from-image` — multipart/form-data with image/PDF file, returns `ImportedMenuData`
- `POST /api/v1/TenantMenus/{ExternalId}/apply-import` — applies imported data with merge strategy
